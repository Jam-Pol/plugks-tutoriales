Wallet

Transfiere saldo como Nequi 💸

⏱ 2 minutos · Nivel: Principiante

Tu Wallet de PLUG-KS no es solo para pagar compras — también puedes enviar y recibir saldo entre usuarios usando QR, ID o número de celular. Rápido, sin complicaciones.

✨ ¿Para qué sirve? Paga a un amigo, divide gastos, recarga el wallet de un familiar o recibe pagos de tus clientes — todo dentro de PLUG-KS.

3 formas de transferir 📲

🔲

Opción 1 — Código QR

Ve a tu Wallet → toca "Transferir" → selecciona "QR" → escanea el QR del otro usuario o muestra el tuyo para recibir.

🆔

Opción 2 — ID de usuario

Ingresa el ID único del usuario al que deseas enviar saldo. Cada usuario tiene su ID en su perfil.

📱

Opción 3 — Número de celular

Ingresa el número celular registrado del destinatario. El saldo llega instantáneo.

Paso a paso para enviar 💰

1

Ve a tu Wallet

Toca el ícono 💳 en la barra inferior de navegación.

2

Toca "Transferir"

Selecciona el método: QR, ID o celular.

3

Ingresa el monto

Escribe cuánto deseas enviar. Verifica que tengas saldo suficiente.

4

Confirma con tu PIN

Ingresa tu PIN de seguridad para autorizar la transferencia.

5

¡Transferencia exitosa! ✅

El saldo llega instantáneamente al otro usuario. Ambos reciben notificación.

🔒 Seguridad: Verifica siempre el nombre del destinatario antes de confirmar. Las transferencias son instantáneas e irreversibles.
